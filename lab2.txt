Заголовок;

Название поля name;
Поле name:
1) Пользователь вводит допустимое имя в поле name;
2) Пользователь вводит недопустимое имя в поле name;
3) Пользователь оставляет поле пустым;
4) Пользователь превысил максимальную длину строки;

Название поля email;
Поле e-mail:
1) Пользователь вводит допустимый email;
2) Пользователь вводит занятый email;
3) Пользователь вводит недопустимый email;
4) Пользователь оставляет поле пустым;
5) Пользователь превысил максимальную длину строки;

Название поля password;
Поле password:
1) Пользователь вводит допустимый пароль;
2) Пользователь вводит недопустимый пароль;
3) Пользователь оставляет поле пустым;
5) Пользователь превысил максимальную длину строки;
6) Пользователь не достиг минимальной длины строки;

Название поля confirm password;
Поле подтверждения пароля:
1) Пользователь вводит отличный от исходного пароль;
2) Пользователь вводит идентичный пароль;
3) Пользователь оставляет поле пустым;

Кнопка зарегистрироваться;
1) Пользователь заполняет форму с одной из комбинаций предыдущих полей и нажимает кнопку;
